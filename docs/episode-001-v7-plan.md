# 第 001 期视频 · 抖音小白优化方案 V7

> 原则：保留 GitHub 录屏场景，其他全部针对小白用户重新设计
> 目标：45-50 秒，前 3 秒有钩子，全程有字幕，术语口语化

---

## 一、改版核心思路

### 三句话策略

1. **前 3 秒戳痛点** — 不展示 Logo/数据，直接一个生活场景
2. **中间讲清楚** — 用类比让小白秒懂"Skills = AI 的 App"
3. **结尾给行动** — 明确告诉用户"打开网址 → 搜 find-skills → 装上"

### 术语翻译表

| 原术语 | 小白版 | 备注 |
|--------|--------|------|
| AI Agent | AI 助手 | 所有人都懂 |
| Skills | AI 技能 / AI 插件 | 类比手机 App |
| find-skills | 技能搜索引擎 | 类比 App Store |
| npx skills add | 复制这行命令安装 | 不解释 npx |
| GitHub Stars | GitHub 获赞数 | 或者直接说"2.5 万人收藏" |
| Vercel | 顶尖科技公司 | 不展开 |
| Claude Code / Cursor | AI 编程工具 | 统称 |
| skills.sh | 技能商店 | 类比 App Store |

## 三、逐段详细方案

### 🎬 段落 1：片头（4 秒，0:00-0:04）

**配音**：（无，留 Jingle 音效）

### 🎣 段落 2：痛点钩子（4 秒，0:04-0:08）

**这是最关键的改动 — 前 3 秒必须抓住小白**

**画面**：
- 左侧：一个人对 AI 说「帮我做个 PPT」
- 右侧：AI 回答「好的，但是…我只能做很基础的…」（灰色文字，表示不够好）
- 中间弹出一个大问号 + 文字「AI 不够专业？」

**字幕条**：`AI 回答太泛？不够专业？`

**配音**：
> "你让 AI 帮你做 PPT，它只会给你一堆大纲模板。"

**设计意图**：戳中每个人用 AI 时"不够专业"的痛点

### 💡 段落 3：一句话价值（3 秒，0:08-0:11）

**画面**：
- 大字弹出：「给 AI 装技能，让它变专家」
- 下方小字：「就像给手机装 App 一样」
- 配一个手机装 App 的简单图标动画

**字幕条**：`给 AI 装技能 = 给手机装 App`

**设计意图**：3 秒内说清楚核心价值，用类比让小白秒懂

## 五、新增：AI 配音旁白

### 旁白脚本（约 150 字，50 秒）

| 段落 | 旁白文字 | 时长 |
|------|----------|------|
| 钩子 | "你让 AI 帮你做 PPT，它只会给你一堆大纲模板。" | 3s |
| 价值 | "今天推荐一个工具，能给 AI 装上专业技能，就像给手机装 App 一样简单。" | 4s |
| GitHub | "这个工具叫 find-skills，GitHub 上 2.5 万人收藏，累计安装 230 万次。" | 4s |
| 演示 | "装好之后怎么用？直接跟 AI 说'帮我做个 PPT'，它会自动搜索、安装最合适的技能，然后直接帮你生成。不用记命令，说人话就行。" | 8s |
| 安装 | "安装只需要复制这一行命令。不管你用 Claude Code、Cursor 还是国内的 Trae，72 种 AI 工具全支持，装法完全一样。" | 6s |
| 结尾1 | "怎么开始？三步：打开浏览器，输入 skills.sh，搜索 find-skills。免费、开源。" | 5s |
| 结尾2 | "你最想让 AI 学什么技能？评论区告诉我。关注我，每天一个 AI 实用工具，明天见。" | 6s |

### 配音技术方案

用 Python 合成语音（已有 ffmpeg + scipy）：
- 方案 A：调用在线 TTS API（如 Azure、阿里云）
- 方案 B：用 `edge-tts`（微软免费 TTS，中文效果不错）
- 方案 C：用 `pyttsx3`（离线，效果一般）

**推荐方案 B**：edge-tts，免费、中文自然、支持语速调节
